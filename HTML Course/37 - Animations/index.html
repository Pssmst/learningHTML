Animations

Notes

  • `animation-iteration-count` is set to `infinite` for display purposes.
  • You can also set `animation-play-state` to `paused` and `running` for obvious reasons.
  • Also see this site to view the various types of `animation-timing-function`.

Movement

  • "from" moves an element from a translated point to its initial point.
  • "to" moves an element from its initial point to a translated point.
  • Percentages move an element in the percentage of the time alloted, then return to the initial position in whatever percentage of time remains.
