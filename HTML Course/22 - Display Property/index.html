Display Property

Block

(All the white space to the right of the div and the block-level span is allocated to those elements.)

div (automatically a block)
span (regular) span (block)

Inline

div (inline)
span (inline) Now that the two elements are inline, they are part of this paragraph.

Inline-Block

div (inline-block)
span (inline-block) These elements are still inline, but can each be specified a width and height. When dealing with inline-block, you can set elements' visibilities to hidden, which is what I did with the div element. This hides the element, although it still is a tangible, interactible thing on the webpage.

None

div (none)
span (none) The none display property literally just erases the elements from existence. There was once a div and span here.
